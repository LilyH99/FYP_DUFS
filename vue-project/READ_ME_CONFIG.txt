Setup first
https://nodejs.org/en/download - go to this website and download Node.js

Create Vue project
go to CMD and paste this 
npm create vue@latest

Click enter on all config options

Enter your project folder
cd <whatever you named the project> then run:
npm install
npm install -D @tailwindcss/postcss
then replace all the files in the project folder with the files in GitHub

Run the Vue project
type in CMD: npm run dev
CTRL + LEFT CLICK on the local host link to open your Vue project
